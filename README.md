# TEMPLATE ADMIN

Template de Gerenciador Web básico

----

## Informações do Projeto

Projeto criado com:
- NextJS V15
- TypeScript
- Tailwind CSS

---- 

## Baixar o projeto

```
git clone https://github.com/CarlosWGama/admin-template.git gerenciador
cd gerenciador
npm install 
npm run dev
```
